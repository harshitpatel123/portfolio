## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Customization

### Personal Information
Update the following in `components/Hero.js`:
- Name and title
- Location
- Description
- Social media links
- Contact information

### Colors and Theme
Modify `tailwind.config.js` to customize:
- Color palette
